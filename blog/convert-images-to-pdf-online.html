Why Convert Images to PDF

Images are great for viewing, but PDFs are better for sharing, archiving, and printing. Here are the most common reasons people convert images to PDF:

  • Document submission — Many institutions (banks, universities, government offices) require documents in PDF format. If you scanned your ID, passport, or receipts as images, you need to convert them to PDF before submitting.
  • Combining multiple files — Sending 15 individual JPEG files as email attachments is messy. A single PDF with all images organized in sequence is cleaner and easier for the recipient to review.
  • Consistent printing — PDFs preserve exact layout, dimensions, and quality across every device and printer. An image file might scale unpredictably, but a PDF prints exactly as intended.
  • Professional presentation — Portfolios, lookbooks, proposals, and reports look more professional as paginated PDF documents than as a folder of image files.
  • Archival — PDF/A is the standard for long-term document archival. Converting scanned images to PDF ensures they remain accessible and searchable for years.

Supported Image Formats

Browser-based converters support all image formats that modern web browsers can render. Here is what works and when to use each format:

Format Best For Notes
JPEG / JPG Photographs, scans Most common format. Lossy compression keeps file sizes small. Ideal for photos.
PNG Screenshots, diagrams, text Lossless compression. Preserves sharp edges and text. Supports transparency.
WebP Modern web images Google's format. Smaller than JPEG at same quality. Good browser support.
BMP Uncompressed images No compression. Very large files. Convert to JPEG or PNG first for smaller PDFs.
GIF Simple graphics Only first frame is used. Limited to 256 colors. Not ideal for photos.
SVG Logos, icons, vector art Rasterized before embedding. For best results, export SVG at high resolution first.
Format Recommendation

For photographs and scans, use JPEG files. For screenshots, UI mockups, and anything with text, use PNG. If you have HEIC files from an iPhone, convert them to JPEG first using your operating system's built-in tools or the Image Compressor.

Tips for Best Quality

Start with High-Resolution Source Images

The quality of your PDF is limited by the quality of your input images. Use the highest resolution available. For print-quality PDFs, images should be at least 300 DPI at the intended print size. For screen-only PDFs, 150 DPI is sufficient and produces smaller files.

Use the Right Format for the Content

JPEGs introduce compression artifacts around sharp edges and text. If you are converting screenshots or documents with text, use PNG source files to maintain crisp edges. If you are converting photographs, JPEG is fine because the lossy compression is designed for continuous-tone images.

Resize Before Converting

If your images are much larger than the target page size (for example, 6000x4000 pixel photos destined for an A4 page), resize them first using the Image Resizer. This reduces the PDF file size without affecting visible quality at the intended viewing size. A 4000-pixel-wide image displayed on a 595-pixel-wide A4 page is wasting data.

Crop Before Converting

Remove unnecessary whitespace, borders, or unwanted areas from your images before building the PDF. The Image Cropper lets you trim images precisely, ensuring each page in your PDF shows only what matters.

Choose Original Quality for Print

If your PDF will be printed, select the original quality setting (no recompression) when converting. This embeds your images at their full resolution. You can always compress the PDF later using the PDF Compress tool if the file size is too large for email.

Watch the File Size

Combining many high-resolution images without compression can produce very large PDFs. A 50-image collection of 5 MB JPEGs could produce a 250 MB PDF. If file size matters, enable compression during conversion or run the output through the PDF Compress tool afterward.

Page Size and Layout Options

Standard Page Sizes

Setting the page size to A4 (210 x 297 mm) or US Letter (8.5 x 11 inches) fits images into a standard document layout. Images are centered on the page and scaled to fit within the margins while maintaining their aspect ratio. This is the right choice for documents you plan to print or submit to institutions that expect standard page sizes.

Fit to Image

Setting the page size to match the image dimensions creates a borderless PDF where each page is exactly the size of its image. No margins, no whitespace, no scaling. This is ideal for photo portfolios, digital presentations, and archives where you want to preserve the exact original dimensions. Each page may have a different size if the source images have different dimensions.

Orientation

Most tools offer three orientation options: portrait (vertical), landscape (horizontal), and auto-detect. Auto-detect examines each image's aspect ratio and sets the page orientation accordingly — landscape images get landscape pages and portrait images get portrait pages. This produces the most natural layout when your image set contains a mix of orientations.
