10 Modules

Everything an AI agent needs.
Nothing it doesn't.

Each module is a precision instrument. Minimal tokens in, maximum value out.

ntool deploy

Deploy

One command to build, deploy, verify SSL, check links, and cache results. Supports Vercel, Netlify, Surge, GitHub Pages, Cloudflare.

ntool check

Check

Verify links, SSL certificates, DNS records, HTTP headers, accessibility, Lighthouse scores. All in one sweep, all in one output.

ntool stats

Stats

Repo metrics, build sizes, dependency health, code coverage, performance benchmarks. Structured JSON output optimized for LLM consumption.

ntool gen

Gen

Generate boilerplate, configs, components, API routes, database schemas. Understands your stack and generates exactly what fits.

ntool memory

Memory

Persistent key-value store that survives sessions. Store decisions, context, progress. Never lose track of what was done or why.

ntool session

Session

Session lifecycle management. Start, checkpoint, resume. Automatic state serialization so any AI agent can pick up exactly where another left off.

ntool scan

Scan

Security scanning, dependency audits, license checks, secret detection. Catch vulnerabilities before they ship.

ntool cache

Cache

Intelligent result caching. Expensive operations are computed once and served instantly on repeat. TTL-aware, context-aware, token-saving.

ntool watch

Watch

File system watcher with intelligent debouncing. Auto-rebuild, auto-test, auto-deploy on changes. Set it and forget it.

ntool config

Config

Auto-detect project stack, tools, and conventions. Zero-config by default, fully customizable when needed. JSON or YAML.

How It Works

Three steps. That's it.

Install

One command. Works on macOS, Linux, and Windows (WSL). No dependencies, no bloat.

curl -sSL ntool.app/install | bash

Configure

ntool auto-detects your stack, tools, and conventions. Zero config needed. Override anything in ntool.json.

ntool config detect

Use

Works with any AI agent: Claude, GPT, Gemini, local models. Just add ntool to the system prompt.

ntool deploy ./my-project
