Privacy Policy

Last updated: February 5, 2026

Overview

NexTool is a browser extension that processes data locally in your browser. We do not collect, store, or transmit any personal data to our servers because we do not have servers.

Data Collection

NexTool does not collect any personal information. Your API keys are stored locally in your browser using Chrome's storage API and are never transmitted to us.

API Requests

When you use NexTool's AI features, your text is sent directly from your browser to the AI provider you configured (OpenAI, Anthropic, or Google). NexTool acts as a client-side interface only. We never see, store, or process your data.

Third-Party Services

NexTool communicates directly with:

  • OpenAI API (api.openai.com) - if you configure an OpenAI key
  • Anthropic API (api.anthropic.com) - if you configure an Anthropic key
  • Google AI API (generativelanguage.googleapis.com) - if you configure a Google key
  • Gumroad API (api.gumroad.com) - for Pro license validation only

Each provider has their own privacy policy governing how they handle your data.

License Validation

If you purchase NexTool Pro, your license key is validated via Gumroad's API. Only the license key is sent, no other personal data.
