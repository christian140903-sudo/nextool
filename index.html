Everything you need to know about Smart Memory and Miguel.

Smart Memory is a Model Context Protocol server that gives AI assistants persistent, intelligent memory. Instead of losing context between sessions, your AI remembers everything — project architecture, coding patterns, decisions, and learnings. It's the difference between an AI that starts from zero every time and one that builds cumulative intelligence.
Chat logs are raw text dumps. Smart Memory uses TF-IDF semantic search, automatic categorization, pattern recognition, and intelligent recall. It doesn't just store — it understands. When you ask "what did we decide about the API design?", it finds the relevant context across hundreds of interactions.
Absolutely. Smart Memory runs entirely on your local machine. Zero data leaves your device. No cloud sync, no telemetry, no external API calls. The entire codebase is open source on GitHub — you can verify every line.
Smart Memory works with any MCP-compatible client: Claude Code, Cursor, Windsurf, and more. The MCP standard is rapidly being adopted across the AI development ecosystem, so compatibility grows every week.
