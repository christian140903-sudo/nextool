Configure your MCP client

Add Smart Memory MCP to your Claude Desktop, Cursor, or any MCP-compatible client. Add this to your MCP configuration file:

// Claude Desktop: ~/Library/Application Support/Claude/claude_desktop_config.json // Cursor: .cursor/mcp.json { "mcpServers": { "smart-memory": { "command": "npx", "args": ["mcp-smart-memory"], "env": { "MEMORY_PATH": "~/.smart-memory" } } } }

After adding the config, restart your MCP client. The memory server will start automatically.
