Specificity Rules: Inline styles (1,0,0,0) beat IDs (0,1,0,0), which beat classes/attributes/pseudo-classes (0,0,1,0), which beat elements/pseudo-elements (0,0,0,1). The universal selector (*) and combinators (+, >, ~) have zero specificity. !important overrides everything.
