When to Use Each Ratio

Minor Second (1.067)

Very subtle differences between levels. Works well for dense UIs, data dashboards, and applications where space is limited.

Major Third (1.25)

A balanced, versatile ratio ideal for desktop body copy. Provides clear hierarchy without excessive contrast between heading levels.

Perfect Fourth (1.333)

The most popular default for web typography. Strong enough hierarchy for marketing pages while remaining practical for content-heavy sites.

Golden Ratio (1.618)

Dramatic and expressive. Best for hero sections, landing pages, and editorial layouts where large headings create visual impact.
