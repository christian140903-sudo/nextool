DNS Record Types Reference

A Record

Maps a domain to an IPv4 address. The most fundamental DNS record type, used to point your domain to a server.

AAAA Record

Maps a domain to an IPv6 address. The modern equivalent of the A record for next-generation internet protocol.

CNAME Record

Canonical name — an alias that points one domain to another. Used for subdomains like www pointing to the root domain.

MX Record

Mail Exchange — specifies which mail servers handle email for the domain, with priority values for failover.

TXT Record

Text records store arbitrary text. Commonly used for SPF, DKIM, DMARC email authentication, and domain verification.

NS Record

Name Server — delegates a DNS zone to authoritative name servers. Defines who controls DNS for the domain.

SOA Record

Start of Authority — contains administrative info about the zone: primary NS, admin email, serial number, timers.

SRV Record

Service record — specifies host and port for specific services like SIP, XMPP, or LDAP with priority and weight.
